Recursos Principais

Sintaxe Clara e Verbosa

Comandos como `create`, `set` e `to` tornam o código legível como se fosse uma frase.

Tipagem Estática Forte

Declare tipos como `integer` e `string` para evitar erros comuns e organizar seu código.

Sistema de Módulos

Organize seu código em arquivos reutilizáveis com `import`, incluindo uma biblioteca padrão (`str`, `math`).

Veja a Prose em Ação

factorial.prose 
# Função com retorno: cálculo de fatorial
function fatorial(integer n) -> integer
    if n == 0 then
        return 1;
    else
        return n * fatorial(n - 1);
    end
end

writeln("O fatorial de 5 é: " + sys.to_string(fatorial(5)));
